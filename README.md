
# Full Stack Employee Management Application Using ReactJS and Spring Boot


## Tech We Used

- ReactJs
- Spring Boot
- MySql Database
- Spring Security
- REST API

## Features

- Login and Logout
- CURD Operations
- Data are stored in MySql database
- Neat and clean UI

## Steps to run in your machine

#### Run the following commands
```

Clone the rep

cd react-frontend
npm i
npm run start

cd springboot-backend
Run as Spring Boot
```
